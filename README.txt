Pour créer les tables dans la base de données : 
    python manage.py migrate
    
Pour créer un superutilisateur :
    python manage.py createsuperuser
    
Avant de continuer, connectez vous sur l'interface administrateur et créez une chatroom :
    https://web-project-forkhe-fishkiller.c9users.io/admin/
    Donnez lui un nom est specifiez bien que son label est"main".
    
Vous pouvez maintenant créer des utilisateurs, vous conecter et poster des messages sur le site !
